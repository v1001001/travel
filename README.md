- `heilongjiang-travel-enhanced.html` - 哈尔滨-漠河11天环线自驾游攻略（增强版）
- `heilongjiang-travel.html` - 哈尔滨-漠河11天环线自驾游攻略（基础版）

# 东北+呼伦贝尔15天精华环线自驾游

## 项目简介

基于多个小红书旅行笔记创建的东北+呼伦贝尔15天精华环线自驾游攻略，总里程5000公里，穿越长白山、大兴安岭、呼伦贝尔草原的史诗级自驾之旅。

## 行程特色

### 🏔️ 长白山区域
- 秋季色彩斑斓的天池
- 壮观的长白山瀑布
- 温泉体验

### 🌲 大兴安岭区域
- 金山鹿苑与梅花鹿亲密接触
- 原始森林穿越
- 界江风光欣赏

### 📍 漠河极地
- 北极村"最北之家"
- 龙江第一湾
- 白桦林风光

### 🌾 呼伦贝尔草原
- 额尔古纳湿地
- 海拉尔草原风光
- 满洲里边境风情

### 🌋 阿尔山火山
- 阿尔山天池
- 火山地貌奇观
- 森林温泉

## 技术特色

- **时间线设计**: 清晰的时间线展示15天行程

# 东三省7天旅行攻略

## 项目简介

这是一个基于HTML5的东三省旅行攻略网页，集成了高德地图API，提供完整的7天旅行规划。

## 功能特色

### 🌤️ 实时天气查询
- 集成高德地图天气API
- 显示哈尔滨、长春、沈阳、大连四地实时天气
- 包含温度、天气状况、风向风力等信息

### 📍 详细行程规划
- 7天完整行程安排
- 每天2个主要景点
- 包含游览时间、门票价格等详细信息

### 🎨 景区卡片展示
- 简约美观的卡片式设计
- 景区图片展示
- 悬停动画效果

### 🚕 交通安排
- 每天景点间的打车安排
- 一键打车功能（集成高德地图打车）
- 显示预计时间和费用

### 🗺️ 地图集成
- 高德地图个人地图链接
- 一键打开高德地图APP
- 完整路线规划展示

### 💰 旅行总结
- 预算估算（门票、交通、住宿、餐饮）
- 旅行注意事项
- 实用建议

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **地图服务**: 高德地图API
- **天气服务**: 高德地图天气API
- **图片服务**: Unsplash

## 使用方法

1. 打开 `kmTravel.html` 文件
2. 查看天气信息和行程安排
3. 点击"在高德地图中打开"查看完整路线
4. 点击各景点的"打车前往"按钮直接打车
